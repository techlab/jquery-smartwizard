## 2024-11-20 - Adding Accessibility to Dynamically Generated jQuery UI Elements
**Learning:** When building jQuery plugins that dynamically generate UI elements (like scroll arrows or navigation buttons), it's easy to overlook accessibility since these elements don't exist in the static HTML markup. For icon-only buttons, they inherently lack accessible names. Additionally, dynamically rendered buttons intended to act as standard navigation tools often miss out on standard hover tooltips.
**Action:** When implementing or modifying dynamically generated interactive UI elements (like `<button>` or `<a>`), always ensure `.attr({ 'aria-label': '...', 'title': '...' })` is used to provide both screen reader accessible names and visual hover tooltips, especially for icon-only components.
